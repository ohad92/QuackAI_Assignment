# Quack AI Home Assignment
Knowledge Base Manager
Basic command-line tool to manage articles for a knowledge base.

Features
1. Add, edit, delete articles.
2. Search by keyword or tag.

Stores articles in a local JSON file.

Setup
1. Clone the repository:
  git clone https://github.com/ohad92/QuackAI_Assignment.git
2. Navigate to the project folder:
  cd QuackAI_Assignment
Run the application:
  python knowledge Base Manager.py

Usage
Follow the on-screen menu to add, edit, delete, or search articles.
